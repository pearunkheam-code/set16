១. សេចក្តីផ្តើមអំពី C++

C++ គឺជាភាសាសរសេរកម្មវិធី (Programming Language) ដ៏មានឥទ្ធិពល និងពេញនិយមបំផុតមួយក្នុងលោក ដែលត្រូវបានបង្កើតឡើងដោយលោក Bjarne Stroustrup។ វាត្រូវបានប្រើប្រាស់សម្រាប់បង្កើត ប្រព័ន្ធប្រតិបត្តិការ (OS), ហ្គេម (Games), និងកម្មវិធីធំៗជាច្រើន។

ឧទាហរណ៍កូដដំបូង (Hello World):

#include <iostream>
int main() {
    std::cout << "Hello, World!";
    return 0;
}
